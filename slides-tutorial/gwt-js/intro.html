GWT components

  • gwt-user:Widgets, ui binder, themes, data table, rpc, internationalization
  • gwt-dev: gwt compiler, codeserver, core utils,...
  • gwt-servlet:servlet for using rpc
  • requestfactory:special client-server mechanism oriented towards data
  • elemental:browser API wrapper
  • jsinterop:annotations that allows to convert js types to java types and vice versa.

GWT compiler

Black box that takes java as an input and produces javascript

it emulates core Java functionalities (e.g ArrayList, java.math, ...etc), full list can be found here: http://www.gwtproject.org/doc/latest/RefJreEmulation.html

Not all Java runtime classes are supported: file API, Threads, Reflection...

Steps

  • Precompile

    defining permutations

    building Java AST

    performing some optimizations

  • Compile

    Convert the Java AST to JavaScript AST

    Perform optimizations and normalizations

    generate javascript code per permutation

  • Linking

    generate a single scripts that links/bootstraps all generated fragments

Precompile -> defining permutations

One permutation per browser * the number of locales

5 permutations: safari (Chrome, Safari, Opera), ie10, ie9, ie8, gecko1_8 (Mozilla)

Precompile -> building the GWT Java AST

Buids a tree structure (abstract syntax tree) that describes the Java code (tokenization, syntax analysis, semantic analysis,..etc)

intermediary representation

Precompile -> perform optimizations

examples of optimizations

  • Pruning: removes globally unreferenced classes
  • Tightening types: making types less general
  • Dead code elimination: removes unused methods and fields
  • Finalizing: making fields and methods final where applicable

Compile

  1. Converts Java AST to JavaScript AST
  2. Pretty much similar but with Js Types instead
  3. runs more normalizations and optimizations
  4. generates the Js source code

Linking

  1. generates a .nocache.js script
  2. boostraper that loads the permutations files
  3. Linkers: CrossSiteIframeLinker (default), D8ScriptLinker, SingleScriptLinker
