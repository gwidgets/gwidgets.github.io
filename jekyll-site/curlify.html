Curlify

Description

Learn how to use curl by using Curlify. Curlify is a UI helper app that generates commands for the famous request building shell libray: curl . Curlify is a pure front end app written in Angular.


Dependencies

  • Angular 5
  • RxJs
  • Angular Material

Where? https://gwidgets.github.io/curlify/
